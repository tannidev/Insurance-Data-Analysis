# Insurance-Data-Analysis
This project performs exploratory data analysis on an insurance dataset to understand factors influencing medical insurance charges. It includes data quality checks, statistical summaries, visualization of numeric and categorical features, and correlation analysis to identify relationships between age, BMI, smoking status, and insurance costs.

---

## Features

* Data quality checks (missing values, duplicates)
* Statistical summary of numeric features
* Distribution analysis of age, BMI, and charges
* Categorical analysis (sex, smoker status, region, children)
* Bivariate analysis of insurance charges
* Correlation analysis and visualizations
* Interactive Streamlit dashboard

---

## Tools & Libraries

* Python
* Pandas
* Matplotlib
* Seaborn
* Streamlit

---

## How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Run the Streamlit app:

```bash
streamlit run app.py
```
